About

Hier kannst du nachlesen, was diese Applikation alles kann.

User

Die Applikation wurde entwickelt für eine WG. Die Verschiedenen Benutzer wurden im vorhinein angelegt und können bis jetzt bloss im Code bearbeitet werden.

Achtung: diese Applikation wurde nicht entwickelt um mehrere WGs anzulgen. Sie wurde spezifisch für eine WG entwickelt.

ToDos

Jeder in der WG kann Ämtlis erstellen.

Jedes Dieser Ämtli hat ein User dem es aktuelll zugewiesen ist. Ausserdem wird die Deadlin ausgerechnet, in der es erledigt sien muss. Dies passiert mithilfe der alten Deadline und den Anzahl intervall Tagen.

Einkauf

In der Applikation kann eine Einkaufsliste geführt werden. Die Einträge könenn von allen erfasst werden und auch abgehakt, sobald etwas eingekauft wurde.

Finanzen

Bei den Finanzen können Rechnungen erstellt werden sobald etwas eingekauft wurde. Es kann eingetragen werden wer alles beteiligt ist und dann wird ausgerechnet, wer wieviel bezahlen muss.
